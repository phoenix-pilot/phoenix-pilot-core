# `vec.h` library unit tests

## File structure

- `vectors_tests.c` contains unit tests of all functions from `vec.h` library:
    - `vec_cmp`
    - `vec_sum`
    - `vec_add`
    - `vec_dif`
    - `vec_sub`
    - `vec_times`
    - `vec_cross`
    - `vec_dot`
    - `vec_len`
    - `vec_normal`
    - `vec_normalize`
